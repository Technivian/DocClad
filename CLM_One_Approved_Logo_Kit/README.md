# CLM One Approved Logo Kit

This package contains code-ready files derived directly from the approved CLM One logo.

## Use these files

### Full horizontal logo

Preferred for the application shell and web:

- `logo/clm-one-logo-transparent.webp`

Fallback and document use:

- `logo/clm-one-logo-transparent.png`

Exact SVG container:

- `logo/clm-one-logo-exact.svg`

### C1 brand mark

Preferred compact asset:

- `mark/clm-one-mark-transparent.webp`

Fallback:

- `mark/clm-one-mark-transparent.png`

### Browser and app icons

Use the files in `icons/` and `site.webmanifest`.

## Important technical note

The approved logo originated as raster artwork. To ensure the code implementation is exactly the approved visual, the supplied SVG files embed the approved raster pixels rather than approximating the wordmark with another font or tracing the symbol.

This is intentional. A newly traced vector would be a new logo and could introduce subtle differences.

## Agent handoff

Give the coding agent:

- this entire folder or ZIP
- `docs/AGENT_APPLY_CLM_ONE_LOGO.md`

The agent must use the supplied artwork and must not recreate the logo.
